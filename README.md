Northshore Logistics Management System

Overview
This project is a database-driven logistics management system developed using Python, SQLite, and Tkinter. It is designed to help Northshore Logistics Ltd manage shipments, deliveries, inventory, vehicles, drivers, and payments through a centralized system.

Features
- Add and manage shipment records
- Assign deliveries with drivers and vehicles
- Track incidents such as delays or damaged goods
- Manage inventory across warehouses
- Add and manage vehicles and drivers
- Record payment details
- Generate reports:
  - Shipment Summary
  - Delivery Report
  - Vehicle Usage
  - Driver Routes
  - Warehouse Activity
  - Low Stock
  - Payment Report

Technologies Used
- Python 3
- SQLite
- Tkinter

Project Structure
- database/ → Database connection and schema
- services/ → Business logic
- security/ → Authentication and encryption
- ui/ → User interface
- logs/ → Log files
- main.py → Application entry point

How to Run the Project
1. Install Python 3
2. Open the project folder
3. Run: python main.py
